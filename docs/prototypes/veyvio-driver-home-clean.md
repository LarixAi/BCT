# Veyvio Driver — Clean Home prototype

Interactive HTML reference was adopted as the Driver **Home** screen (`CleanHomeScreen`).

## Hierarchy (shipped)

1. Operational status strip (tone by lifecycle)
2. **Brand** — VEYVIO / DRIVER lockup + product name (on light)
3. Headline + context
4. Map-led overview
5. One primary action card
6. Flat **Today** rows (vehicle, messages, actions, schedule)
7. App shell bottom nav (Home · Duties · Checks · Messages · More)
